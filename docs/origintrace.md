# OriginTrace

**Geographic-Indication coffee traceability for the Lake Kivu GI Association.**

## Overview
Geographic-Indication (GI) status protects the identity and value of a region's coffee, but
only if origin and production can be verified end to end. OriginTrace provides the traceability
backbone for the Lake Kivu GI Association, from farm to certified origin.

## What it does
- Public landing and information platform for the GI Association
- **Phase 2 — Agronomy:** agronomic and production monitoring for producers
- Mobile data capture in the field, backed by an Odoo operations backend
- Traceability records supporting Geographic-Indication certification

## Stack
