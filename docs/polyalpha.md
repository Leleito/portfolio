# PolyAlpha

**Algorithmic trading system for prediction markets.**

## Overview
PolyAlpha is a systematic trading system for prediction markets. It combines signal generation
with disciplined, risk-managed position sizing to trade methodically rather than
discretionarily.

## What it does (high level)
- Systematic signal generation from market data
- Liquidity-adjusted, risk-managed position sizing
- Backtesting and paper-to-live evaluation workflow

> Strategy details, signal logic and source code are **private and proprietary**. This page is
> a high-level overview only.
