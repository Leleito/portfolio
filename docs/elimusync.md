# ElimuSync

**School ICT-readiness map for Kenya's ~24,000 primary schools.**

## Overview
Planning national digital-learning infrastructure needs a clear picture of which schools are
connected, powered and equipped. ElimuSync turns fragmented school data into an interactive
map that supports evidence-based rollout decisions.

## What it does
- Interactive map of all 47 counties with schools plotted by connectivity status
- Per-school detail: ICT-readiness score, enrolment, device count, power and connectivity type
- County filters and a KPI dashboard (% connected, % with devices, % assessed)
- Layer toggles for connectivity, devices, power and pilot programmes

## Stack
TypeScript · React · Leaflet · Tailwind CSS · Vercel
