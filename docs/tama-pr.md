# Tama PR

**Ethical PR & communications consultancy.**

## Overview
Tama PR is an ethical public-relations and communications consultancy. Its site presents the
practice, its principles and its services to prospective clients.

## What it does
- Consultancy website and brand presence
- Content managed for easy editorial updates
- Clear presentation of services and approach

## Stack
Astro · Tailwind CSS · Sanity.io · Vercel
